# Tea_Plantation_Managemt

## Development

**1. clone code**

git clone [https://github.com/dmcgit/Tea_Plantation_Managemt.git](https://github.com/dmcgit/Tea_Plantation_Managemt.git).

**For Frontend**

**2. install**
```
cd **
npm install
```
**3. run app**
```
npm start
```
**For Backend**

**4. install**
```
cd **
npm install
```
**5. run app**
```
npm start
```
([if you use nodemon(suggest to use), 'nodemon start'])


## Technologies
Frontend - Reactjs 

Backend - Nodejs 

Database - Mysql
